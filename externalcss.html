External css properties

In external css type, we link the css file in which
we define styles of the webpage and link that css file
with the main html document. The advantage of external
css type is that

  1. mono style can be applied to multiple
    web pages
  2. If the content of the css file and
    html file is too long ,then it is better to separate two files
    and link for more clear and and precise view ,also it make
    easier for the server-side programmers to edit(if need)
